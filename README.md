# Teste Wine

## [Demo](https://luhoff-wine.netlify.app/#/)

## Instalação do projeto
```
npm install
```

### Compilar para desenvolvimento (com hot-reloads) 
```
npm run serve
```

### Compilar e minificar para produção  
```
npm run build
```

### Realizar testes unitários
```
npm run test:unit
```

### Rodar lints
```
npm run lint
```
